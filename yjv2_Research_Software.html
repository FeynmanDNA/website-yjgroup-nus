Software

People who are really serious about software should make their own hardware.

LabVIEW program to control magnetic tweezers

Our group has a series of customized LabVIEW programs for tracking magnetic particles both in vertical and transverse magnetic tweezers. The main contributors for the core of the software are Chen Hu, Lim Ci Ji, and Yan Jie. Other contributors over the years are Yao Mingxi, Lu Chen, Ricksen Winardhi, Ryo Kawamura, and Zeng Xiangjun. Currently Lu Chen is developing a new version to control the motor for transverse magnetic tweezers. And Zeng Xiangjun is working on a drug-screening automation machine, based on the vertical magnetic tweezers and the LabVIEW program. We welcome discussions on LabVIEW development for biophysics research, and the basic version of the source code is available upon request.

Matlab program to determine persistence length of DNA

Cong Peiwen wrote the matlab program to determine the centreline of DNA and thus able to calculate the persistence length of DNA. This matlab program has been used to measure DNA persistence length from AFM images both in Yan Jie's teaching course, as well as in the Soft Matter paper published in 2014. The source code of the Matlab script is available for download here.

Python-OpenCV script to measure rotation angle of asymmetric magnetic particles

In order to determine the rotation angle of an asymmetrical magnetic particles, for example, a dimer of two different sizes, Ryo Kawamura wrote a Python script based on the OpenCV libraries. The Python code uses eigenvector from Principle Component Analysis in OpenCV library. The script is available for download here.
